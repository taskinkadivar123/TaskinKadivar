Taskook Salon

Python · Django · SQLite | Booking • AJAX • Django


Overview

A salon booking and management platform built with Django. Allows customers to book appointments online with a service selector and full AJAX integrations, while sending email confirmations on booking actions.

The Problem

Salon businesses need a lightweight booking system that allows customers to schedule appointments online, check real-time availability, and receive confirmation emails.

The Solution

Developed a Django backend application with email routing integration, service management, and customer appointment booking modules loaded via AJAX to avoid page refreshes.

Architecture

Django MVT architecture with integrated SMTP email delivery backend and modular appointment slots scheduling logic.

Database Design

SQLite with tables for Users, Service, Appointment, TimeSlot, and Notifications.

Technical Challenges

▹ Building real-time booking availability check inputs
▹ Configuring transaction email routing on booking updates
▹ Managing appointment slot conflicts during peak hours

Tech Stack

Python Django SQLite HTML CSS JavaScript AJAX REST API Email Integration

▹ Email Integration
▹ Service Management Catalog

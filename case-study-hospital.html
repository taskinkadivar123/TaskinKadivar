PHP · MySQL | Dashboard • Hospital


Overview

A comprehensive clinic management platform built inside WordPress. Please note: this web portal is optimized for Microsoft Edge browser configurations.

The Problem

Hospital clinics need a portal for patients to sign up, change credentials, review hospital information, and manage booking history.

The Solution

Built a clinic dashboard inside WordPress with membership plugin configurations, SMTP email delivery, and user profiles.

Architecture

WordPress Content Management System running PHP with appointment plugins and SMTP configurations.

Database Design

MySQL database hosting user records, appointments history logs, metadata, and pages.

Technical Challenges

▹ Configuring custom secure membership dashboards inside WordPress
▹ Restricting appointment actions to verified logged-in users
▹ Edge browser specific plugin dependencies

Overview

A full-featured e-commerce platform for digital books, built with Django and Python. The platform allows users to browse, purchase, and read digital books online with a complete shopping experience.

The Problem

Readers need a centralized platform to discover, purchase, and access digital books with a seamless user experience — including wishlists, carts, and order management.

The Solution

Built a comprehensive Django-based e-commerce application with user authentication, product catalog, shopping cart, checkout flow, order tracking, blog section, and a powerful admin dashboard for managing products, orders, and reports.

Architecture

Django MVT (Model-View-Template) architecture with SQLite database. Organized into modular Django apps for users, products, orders, and blog functionality.

Database Design

SQLite with Django ORM. Models include User, Product (Books), Category, Cart, CartItem, Order, OrderItem, Wishlist, Blog, and Review.

Technical Challenges

▹ Implementing a seamless checkout flow with cart management
▹ Building a responsive book reader interface
▹ Creating a comprehensive admin dashboard with analytics
▹ Handling user sessions and authentication securely
